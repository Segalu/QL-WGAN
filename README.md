# BattleshipRL.py
Reinforeced Learning version of the Battleship game, it is a standalone training and inference, the weights are saved as **battleship_qtable.pkl**

# BattleshipGAN.py
WGAN version of the Battleship, this is just the inference or game, the training algorythm is made with the **WGAN-GP-train_shot_gan.py**, and the generated weights are saved on the **models** subdirectory
